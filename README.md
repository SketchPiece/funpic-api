# FunPic Api

> Simple api to create funny pictures

## Installation

```bash
# backend dependencies
$ npm install
```

```bash
# docs dependencies
$ npm run docs:install
```

## Running the app

```bash
# backend start
$ npm run start

# backend start for production
$ npm run start:prod

# build docs for production
$ npm run docs:build

# backend watch mode
$ npm run start:dev
```
